InfiniTrack — Installation

01

Download Source Files

Download all files from Google Drive: infinitrack.rar, flutter_db.sql, PMFLUTTERAPP.rar

Open Google Drive
02

Import Database

Open phpMyAdmin, create a new database named flutter_db, then import flutter_db.sql.

03

Setup Back-End (PHP)

Extract PMFLUTTERAPP.rar and place it in:

C:\xampp\htdocs\PMFLUTTERAPP\infinitrack\
04

Extract Flutter Project

Extract infinitrack.rar to any folder, e.g. C:\infinitrack\

05

Build APK

Open the infinitrack folder in VS Code, then run:

flutter build apk --release

APK output: build/app/outputs/apk/release/InfiniTrack-v0.1.0.apk

06

Access Web Panel

Start XAMPP (Apache + MySQL), then open:

http://localhost/PMFLUTTERAPP/infinitrack/login.php
